## Overview

Interactive prototypes comparing two billing models for Dialpad's AI Agent product:

- **Scenario A**: Upfront credits with budget allocation
- **Scenario B**: Dynamic tiered rates with progressive discounts

Both scenarios include:
- Interactive usage tracking
- Budget/spend management
- Usage history with filtering
- Comprehensive FAQ sections
- Notes page with PRD requirements and decision framework
